# Roles Module Guidelines

## Scope
Applies to all files under `src/roles/`.

## Purpose
Role entities and module wiring to support role-based access.

## Conventions
- Keep role entities in `entities/`.
- Reuse shared enums from `src/common/enums`.
- Document any new role types or relationships here.

## Key Files
- `roles.module.ts`
- `entities/`
